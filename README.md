# Product-Optimization-Revenue-Contribution-Analysis-for-Afficionado-Coffee-Roasters
Power BI dashboard for product optimization, revenue contribution analysis, Pareto analysis, and business recommendations using coffee shop sales data.
## 📊 Key Insights
Focus investment on high-revenue products and categories.
Optimize staffing and inventory based on time-slot demand.
Expand best-performing product strategies across all store locations.
Use Pareto Analysis to prioritize marketing and inventory decisions.
Continuously monitor product performance to improve profitability and customer satisfaction.
## Tools & Technologies
Power BI
Power Query
DAX
Microsoft Excel
